####一. HTML简介
>1.  什么是HTML？
HTML是用来描述网页的一种语言
* HTML是指超文本标记语言
* HTML不是一种编程语言，而是标记语言
* HTML使用标记标记标签来描述网页

> 2.  HTML标签
*  HTML 标签是由尖括号包围的关键词，比如 <html>
* HTML 标签通常是成对出现的，比如 <b> 和 </b>
* 标签对中的第一个标签是开始标签，第二个标签是结束标签
* 开始和结束标签也被称为开放标签和闭合标签

>3. HTML文档=网页
* HTML 文档描述网页
* HTML 文档包含 HTML 标签和纯文本
* HTML 文档也被称为网页

####二. HTML文档的结构

	<html>
		<head>
			头部内容：定义标题式样等
		</head>
		<body>
			主题内容：网页要显示的各种信息，包括文本超链接，图像，动画等
		</body>
	</html>
